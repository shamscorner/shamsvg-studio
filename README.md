# SHAMSVG Studio

A small Svelte + TypeScript + Vite application for creating and experimenting with SVG graphics.

## Quick Start

- Install dependencies: `npm install`
- Run development server: `npm run dev`
- Build for production: `npm run build`
- Preview production build: `npm run preview`

Contributions are welcome. Fork the repo, create a branch, make changes, and open a PR against `main`.
